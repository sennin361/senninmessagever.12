# 🧙‍♂️ 仙人チャット（Sennin Chat）

リアルタイム通信・画像/動画/音声送信・セキュア通信・PWA対応の完全自作チャットアプリです。

---

## 🔥 主な機能

- 🗨️ リアルタイムチャット（Socket.IO）
- 🖼️ 画像・🎥 動画・🎤 音声ファイル送信
- 🔐 AES暗号化通信
- ✅ 既読機能（メッセージ既読ステータス表示）
- 🎨 見やすいUI（吹き出し型、色分け）
- 🧩 PWA対応（オフライン利用・ホーム画面アイコン）
- 🚀 Render対応サーバー構成

---

git clone https://github.com/yourusername/sennin-chat.git
